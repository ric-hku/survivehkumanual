# 2025-26 年度住宿费

## 住宿费

### 住宿舍堂（Residential Halls）

* **校内（李国贤堂、太古堂）；一村（何东夫人纪念堂、施德堂）；二村（李兆基堂、马礼逊堂、孙志新堂）；沙宣道（利铭泽堂、伟伦堂、利希慎堂）；大学堂：**\
  单人间：HKD 31,161 / 年；\
  双人间：HKD 23,805 / 年。
* **圣约翰学院：**\
  小型房间（Small Student Room）：HKD 8,250 / 月；\
  大型房间（Large Student Room）：HKD 9,150 / 月。
* **利玛窦宿舍：**\
  HKD 24,395 / 年。

### 住宿学院（Residential Colleges）

* **三村（信兴学院、志新学院、立之学院、日新学院）：**\
  单人间：HKD 38,490 / 年；\
  双人间：HKD 29,183 / 年。
* **四村（黄竹坑）：**\
  单人间：HKD 48,776 / 年。

{% hint style="warning" %}
以上价格（除圣约翰学院、利玛窦宿舍）**只适用于本科生（UG）**，不适用于研究生（PG）。
{% endhint %}

### 其他

* **青莲台学生宿舍：**\
  双人间：HKD 7,081 / 月；\
  单人间（高架床）：HKD 7,558 / 月；\
  单人间：HKD 8,204 / 月；\
  单人大床间：HKD 8,953  / 月；\
  单人大床间（独立卫浴）：HKD 9,906 / 月。
* **学生公寓（Student Flats）：**\
  HKD 19,977 / 年。

## 强制性餐费

* **大学堂：**\
  HKD 9,700 / 年。
* **利玛窦宿舍：**\
  HKD 10,600 / 年。
* **圣约翰学院：**\
  HKD 11,100 / 年。

{% hint style="info" %}
其他宿舍**没有强制性餐费**。
{% endhint %}
