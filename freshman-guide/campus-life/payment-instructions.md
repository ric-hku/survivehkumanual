# 缴费指南

## 一、确认**付款通知与**金额

进入 HKU Portal，在左侧菜单栏点击 **Financial Services > My Invoice**，查看**付款通知（Invoice）**。

付款通知上会显示**账单类别（Bill Type）**&#x548C;**账单户口号码（Bill Account Number, Bill A/C No.）**：

* **账单类别（Bill Type）**：
  * **‘01’ 学费 Composition Fee**、罚款 Surcharge
  * **‘03’ 住宿费 Hall Fee**
  * ‘05’ 贷款还款 Student Loan Repayment
  * ‘06’ 其他收费 Other Fees
* **账单户口号码（Bill Account Number, Bill A/C No.）**：
  * 14 位数字。除贷款还款外，账单户口号码是**学生证号码（303......）+ “9999”**。

## 三、常见问题与解答

### Q1. 什么时候需要交学费 / 住宿费？是一起交吗？

学费和住宿费一般**按学期为单位**交。学校会在每学期需要缴费的时候，发送**付款通知（Invoice）**&#x5230;学生的**学校电子邮箱**（HKU Connect）。

{% hint style="info" %}
如果在 **HKU Portal** 中提供了**香港电话号码**，会同时收到**短信**通知（发送人：HKU\_FEO）。
{% endhint %}

一般的**缴费时间**如下：

* **第一学期**：
  * **学费**：9 月中旬发送付款通知；截止日期为 9 月 30 日。
  * **住宿费**：9 月中下旬发送付款通知；截止日期在大约一个月后。
* **第二学期**：
  * **学费**：1 月上旬发送付款通知；截止日期为 1 月 31 日。
  * **住宿费**：12 月 – 1 月发送付款通知；截止日期在大约一个月后。

具体金额请见下文 [“Q4. 我要交多少学费 / 住宿费？”](payment-instructions.md#q4.-wo-yao-jiao-duo-shao-xue-fei-zhu-su-fei)。

需要注意的是，学费与住宿费的**账单类别（Bill Type）不同**：学费（及其他费用）为 “01”，住宿费为 “03”。因此，缴费时需要选择 / 输入**正确的账单类别**。

### Q2. 学费 / 住宿费交晚了，会怎么样？

根据《总规例》（[General Regulations](https://calendar.hku.hk/general-regulations/)）第 G17 条：

* 未在**截止日期（Due Date）后 30 天内**支付费用的学生将被注销（De-registered），并**禁止使用大学设施和服务**。
* 若要恢复学生身份，学生必须支付**未付费用的 10%** 或 **HKD 500** 作为**附加费**（**以较低者为准**）。
* 此外，学生可能需要赔偿大学因任何逾期付款或未付款项而产生的任何利息损失。

{% hint style="info" %}
如果在**截止日期（Due Date）后 30 天内**支付费用，**不会产生负面影响**。
{% endhint %}

{% hint style="warning" %}
如果无法在截止日期前支付学费，必须在截止日期前联系**学生发展与资源中心（CEDARS）**&#x7533;请延期支付。

详情请见：[Fee Deferment](https://www.cedars.hku.hk/campuslife/finances/fee-deferment)。
{% endhint %}

### Q3. 我有奖学金，可以不交学费吗？

对于有奖学金的同学，**奖学金**通常**直接支付到学生的银行账户**（同时会向学生的大学邮箱发送通知邮件）。因此，同学需要在香港自行选择一家银行开设储蓄 / 活期账户，并在 HKU Portal 中输入银行账户信息。

因此，可以等奖学金**到账之后**再进行缴费；但不能不缴费。

如果奖学金不能在截止日期前到账，可以向财务及企业管理处（FEO）说明情况。或者，**在截止日期前**通过学生发展与资源中心（CEDARS）申请**晚交学费（**&#x8BE6;见 [Fee Deferment](https://www.cedars.hku.hk/campuslife/finances/fee-deferment)）。

### Q4. 我要交多少学费 / 住宿费？

**2026-27 年度**的**学费**如下：

* 本地学生：HKD 47,000 / 年
* 非本地学生（非 STEM）：HKD 224,000 / 年
* 非本地学生（STEM）：HKD 249,000 / 年

{% hint style="info" %}
**STEM**：牙医学院、工程学院、李嘉诚医学院、理学院；生物医学工程学院、计算与数据科学学院（包括与文学院合办开设的 STEM 相关双学位课程）、创新学院；

**非 STEM**：建筑学院、文学院、经管学院、教育学院、法律学院、社会科学学院；未来媒体学院。
{% endhint %}

一般来说，非本地学生的学费是以入学年度的为准；本地学生的则以当下年度的为准。也就是说：如果将来学费发生变化，**非本地学生的学费保持与入学年度的一样**；**本地学生的则随之变化**。

{% hint style="info" %}
有关**本地学生**的定义，请参阅 [本地及非本地学生的定义](../../appendices/definition-local.md)。
{% endhint %}

**学费**（及其他费用）的缴费安排如下：

* **留位费**，在**接受录取时**缴纳；
* **第一部分**（1st Instalment），在**第一学期**缴纳：
  * 第一学期学费（Composition Fee），减去留位费（Admission Deposit）：
    * 本地学生：HKD 23,500 减去留位费
    * 非本地学生（非 STEM）：HKD 112,000 减去留位费
    * 非本地学生（STEM）：HKD 124,500 减去留位费
  * 学生活动费（Student Activity Fee）：HKD 100
  * 保证金（Caution Money）：HKD 350
  * 录取行政费（Admin Fee for Admission）：HKD 3,000
* **第二部分**（2nd Instalment），在**第二学期**缴纳：
  * 第二学期学费（Composition Fee）：
    * 本地学生：HKD 23,500
    * 非本地学生（非 STEM）：HKD 112,000
    * 非本地学生（STEM）：HKD 124,500

{% hint style="warning" %}
如果发现 HKU Portal 里的需缴纳金额、这里所列的金额、学校电邮所发 Invoice 中的金额**有所出入**，建议**以 HKU Portal 里的为准**。

必要时，请咨询财务及企业管理处（FEO）：[feo\_student@hku.hk](mailto:feo_student@hku.hk)。
{% endhint %}

不同宿舍的住宿费不同，具体请参阅 [2025 – 26 年度住宿费](../hall-selection/2025-26-lodging-fees.md)。

### Q5. 我多交了钱，能退款吗？

**注意**：已缴纳的费用**不予退还**。

如果多交了学费 / 住宿费，多交的部分**会留在学生账户中**；下次缴交费用时，这部分钱仍可以用于支付对应账单类型的费用（也即，**可以少交这部分**）。

如有特殊情况需要特殊考虑减免/退还学费，请将申请发送至 [fees@hku.hk](mailto:fees@hku.hk)，并提供以下信息以加快处理：

* 英文姓名、学生证号（University Number）及所学课程（Curriculum）；
* 涉及豁免 / 退款申请的费用类型和金额；
* 一份详细说明可证明特殊情况的陈述（相关证明文件以附件形式提供）。

### Q6. 我如何知道学校已收到费用？

确认收到费用后，学校将在 3 – 4 个工作日内，将缴费记录（Payment History）和收据（Receipt）更新在 **HKU Portal 中**。在 **Financial Services > My Receipt** 里可以查看。

同时，一般也会发送邮件到学校邮箱，确认已收到费用。

如果一周后学生账户仍未显示缴费记录，请联系财务及企业管理处（FEO）：[feo\_student@hku.hk](mailto:feo_student@hku.hk)。

### Q7. 缴费时 / 缴费后遇上问题怎么办？

如在缴费后遇上问题，请**详述问题之所在**及**附上付款收据副本或网上银行 / 手机付款记录截图**，以电邮发送至：[feo\_student@hku.hk](mailto:feo_student@hku.hk) 咨询。

{% hint style="info" %}
如有其他问题，也可以阅读学务及考试部（ASE Section）关于费用的常见问题：[Frequently Asked Questions](https://ase.hku.hk/fees/faq.html)
{% endhint %}
